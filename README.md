## "Cook With Me"

#### Site internet de partage de recettes de cuisine

Après avoir cloné le projet, installez les dépendances :  
`npm install`

Pour lancer le serveur sur le port 5003 :   
`npm start`

Pour la partie "Front" :  
`cd client`  

`npm install`  

`npm start`
